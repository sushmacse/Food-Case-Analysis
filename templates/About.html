🍽️ About the Project

Comprehensive Analysis and Dietary Strategies with Tableau: A College Food Choices Case Study

📖 Project Overview

This project focuses on understanding the dietary habits, nutritional preferences, and food consumption patterns of college students. Using Tableau, interactive dashboards and stories are created to visualize meaningful insights, helping identify trends in healthy and unhealthy eating behaviors.

🎯 Problem Statement

College students often have diverse eating habits influenced by lifestyle, academic schedules, and personal preferences. This project analyzes food choice data to identify patterns, compare nutritional preferences, and provide valuable insights through interactive Tableau visualizations.

🎯 Objectives

  • ✔ Analyze students' food choices.
  • ✔ Study healthy and unhealthy eating habits.
  • ✔ Identify nutritional trends.
  • ✔ Build interactive Tableau dashboards.
  • ✔ Create an engaging Tableau Story.
  • ✔ Integrate Tableau with Flask.
  • ✔ Deploy a responsive web application.

🛠 Technology Stack

Python Flask Tableau Public SQLite HTML5 CSS3 JavaScript Render
